# Scan Master: Trozo de nube de puntos del panel de acero

Este conjunto de datos contiene un fragmento de la nube de puntos
obtenida mediante el escaneo 3D del panel de acero con el escáner
llamado **Master**. Cada fila representa un punto en el espacio con sus
coordenadas X, Y y Z.

## Format

A matrix with 6955566 rows and 3 columns:

- X:

  Coordenada X de cada punto

- Y:

  Coordenada Y de cada punto

- Z:

  Coordenada Z de cada punto

## Source

Datos de escaneo 3D de un panel de acero
